# Laniakea

Belajar bahasa Nim.

Untuk menjalankannya,

1. Compile Berkas .nim

   ```bash
   nimble c berkas.nim
   ```

2. Nanti dihasilkan objek (exe/binary)

   ```bash
   ./berkas
   ./berkas.exe
   ```

---

Catatan :

- Compile dengan nama tertentu lakukanlah dengan,

```nim
nim c --out:conditional a8conditional.nim
```
